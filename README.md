## 1. 프로젝트 개요
<b> 주제 : 시각장애인과 휠체어이용자들이 키오스크를 사용할 때 발생하는 문제 해결 </b>
- 키오스크 이용자 각각의 눈높이 맞춤 화면 높이 제공 
- 메뉴 선택부터 결제 마지막 단계까지 음성 출력 서비스와 점자 버튼으로 주문할 수 있는 서비스   제공
- 앱을 통해 사업자(관리자)가 메뉴를 관리하고 매출을 조회할 수 있는 서비스 제공
